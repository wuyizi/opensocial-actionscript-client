Classpublic class AlbumsRequestOptions
InheritanceAlbumsRequestOptions RequestOptions

Options for osapi.albums service. The options format is based on the OS-Lite style. The key names of the options object is defined in the protocol spec.

It's used to initialize the AsyncDataRequest instance.
