Classpublic class MyspacePhotosRequestOptions
InheritanceMyspacePhotosRequestOptions RequestOptions

Experimental options for MyOpenSpace photos service. It covers these request types in myspace: MyOpenSpace.RequestType.FETCH_PHOTO and MyOpenSpace.RequestType.FETCH_PHOTOS. The options format is based on the OS-Lite style. The key names of the options object is defined in the protocol spec.

It's used to initialize the AsyncDataRequest instance.
