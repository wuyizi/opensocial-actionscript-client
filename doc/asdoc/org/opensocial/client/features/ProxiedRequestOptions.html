Classpublic class ProxiedRequestOptions
InheritanceProxiedRequestOptions RequestOptions

Options for makeRequest service. The options format is based on the OS-Lite style. The key names of the options object is defined in the protocol spec.

It's used to initialize the ProxiedRequest instance.

NOTE: Currently this feature is located at osapi.makeRequest, but in next spec version, it will be as the osapi.http service.
